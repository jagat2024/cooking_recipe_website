# 🍽️ Jagat Di Sweets

A simple and visually appealing food website built using **HTML and CSS**.
This project showcases traditional Indian sweets like **Gulab Jamun** and **Barfi** with a clean layout, modern UI, and interactive hover effects.

---

## 🚀 Features

* 🧾 Structured layout using semantic HTML
* 🎨 Beautiful UI with gradients and card design
* 🖼️ Image section with hover zoom effect
* 🔥 Overlay effect ("Click here to buy") on images
* 📱 Clean and responsive-like spacing
* 🧭 Navigation bar and footer included

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Hover Effects, Gradients)
